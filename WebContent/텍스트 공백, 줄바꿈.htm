CSS3 텍스트의 공백 처리 연습

(white-space:normal) normal은 연속해서 나오는 공백이나 탭 그리고 줄 바꿈을 모두 하나의 공백으로 처리 그리고 박스 가로 폭에 따라서 자동 개행

(white-space:nowrap) nowrap은 연속된 공백이나 탭을 하나의 공백으로 처리

(white-space:pre) pre값은 연속된 공백( )이나 탭{ } 그리고 줄 바꿈을 모두 처리하지 않고 원문 그대로 표시

(white-space:pre-wrap) pre-wrap은 연속하는 공백( ) 문자들을 하나의 공백으로 처리하지 않지만 줄 바꿈은 박스의 가로 폭을 넘는 경우에 처리

(white-space:pre-line) pre-line은 줄 바꿈한 것이 적용되면서 박스의 가로 폭을 넘어가는 텍스트도 자동으로 개행
